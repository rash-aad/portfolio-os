PROJECT 2: Movie Recommendation System
===================================================================
Repository: https://github.com/rash-aad/Movie-recommendation-system
Language: Jupyter Notebook (99%), Python (1%)
Stars: 1

Description:
A content-based movie recommendation system built using Python,
Pandas, and Streamlit. Recommends movies based on similarity of
content features such as genres, keywords, cast, and crew metadata.
Powered by TMDB 5000 Movie Dataset.

Features:
- Recommends top 5 similar movies based on selected input
- Web app built with Streamlit for intuitive UI
- Cosine similarity on processed metadata
- Fast and responsive with precomputed similarity matrix
- Content-based filtering (not collaborative)

How It Works:
1. Dataset: Uses tmdb_5000_movies.csv and tmdb_5000_credits.csv
2. Preprocessing: Cleans and combines genres, overview, cast, crew into "tags" feature
3. Vectorization: Applies CountVectorizer for word-frequency matrix
4. Similarity Calculation: Cosine similarity across all movie vectors
5. Web App: Streamlit loads pre-trained pkl files for instant recommendations

Tech Stack:
- Python, Pandas, Streamlit
- Scikit-learn (CountVectorizer, cosine similarity)
- Jupyter Notebook for model development
- Pickle files for model serialization (movies.pkl, similarity.pkl)

Data Source: TMDB 5000 Movie Dataset (Kaggle)
